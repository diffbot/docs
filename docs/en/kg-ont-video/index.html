Video

The Video entity type encompasses videos found throughout the web and known to the Knowledge Graph.

Note that fields are not guaranteed to exist in every entity record.

New to the Diffbot Knowledge Graph? Start here.

Video Fields

  • author
  • date
  • duration
  • embedUrl
  • html
  • images
  • language
  • mime
  • naturalHeight
  • naturalWidth
  • text
  • title
  • url
  • viewCount

Video Field Details

Note that certain longer field examples may be truncated for readability.

author

  • Type: String
  • Example:
{
    "author": "Io-Tahoe"
}

date

  • Type: DDateTime
  • Example:
{
    "date": {
        "str": "d2018-12-12T00:00",
        "precision": 4,
        "timestamp": 1544572800000
    }
}

duration

  • Type: Integer
  • Example:
{
    "duration": 2964
}

embedUrl

  • Type: String
  • Example:
{
    "embedUrl": ""
}

  • Type: GlobalIndexImage
  • Example:
{
    "images": [
        {
            "summary": "",
            "image": "https://i.ytimg.com/vi/-w6blCJ0CTY/maxresdefault.jpg",
            "name": "Deconstructing Data Complexity and the Case for AI-Driven Smart Data Discovery",
            "title": "Deconstructing Data Complexity and the Case for AI-Driven Smart Data Discovery",

language

  • Type: String
  • Example:
{
    "language": "en"
}

mime

Indicates the nature and format of a document; MIME type, if available, as specified by the Video's `Content-Type.`

  • Type: String
  • Example:
{
    "mime": "video/mp4"
}

  • Type: Integer
  • Example:
{
    "naturalHeight": 720
}

naturalWidth

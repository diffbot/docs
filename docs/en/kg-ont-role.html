Role

The Role entity type encompasses all roles known to the Knowledge Graph.

Note that fields are not guaranteed to exist in every entity record.

Role Field Details

fields

Specify optional fields to be returned from any fully-extracted pages, e.g.: &fields=querystring,links.

  • Type: com.diffbot.kg.fields.LinkedEntity
  • Example:
{
    "fields": "[See Role]"
}
